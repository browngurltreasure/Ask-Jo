## 🎯 About

**Ask Jo** is a free, web-based AI assistant designed specifically for young people (ages 15-35) in Cameroon and across Africa. It provides instant, personalized help with:

- 🔍 **Job hunting** — local and remote opportunities
- 📝 **CV & cover letter writing** — complete documents, not just tips
- 🎓 **Scholarships & grants** — current deadlines and applications
- 💡 **Free training** — ALX, Google Certificates, Coursera, and more
- 🚀 **Entrepreneurship** — business plans, startup registration
- 📚 **School subjects** — math, science, languages, GCE/BAC prep
- 💻 **Coding help** — Python, JavaScript, HTML/CSS, debugging
- ❤️ **Mental health** — motivation, stress management, relationships
- 🌐 **Translation** — English ↔ French and other languages

> **No account needed. Completely free. Always here for you.**

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🤖 **AI-Powered** | Uses Groq's Llama 3.3 70B model — fast, smart, free |
| 🌍 **Web Search** | Automatically searches the web for current jobs, scholarships, and news |
| 🇬🇧🇫🇷 **Bilingual** | Switch between English and French with one click |
| 📱 **Responsive** | Works perfectly on mobile, tablet, and desktop |
| 🌙 **Dark Mode** | Automatically adapts to your system theme |
| ⚡ **One-Click Topics** | Pre-made buttons for common questions |
| 💬 **Conversation Memory** | Remembers last 40 messages for context |
| 🔗 **Clickable Links** | URLs and references become clickable |
| ✍️ **Markdown Support** | Bold, italic, and formatted text |
