# Ask-Jo
Ask Jo — Youth Opportunity Counsellor  An AI-powered assistant for youth in Cameroon and Africa. Helps with jobs, CVs, scholarships, training, business, school subjects, mental health, and life advice. Free, bilingual (EN/FR), with automatic web search for current opportunities.  Tech: HTML/CSS/JS + Groq AI (Llama 3.3) + Web Search API
